Benutzung des AdInspector

Da der **AdInspector** vollständig clientseitig im Browser operiert, kann er ohne Installation von Browser-Erweiterungen direkt als **Bookmarklet** (Lesezeichen-Skript) eingerichtet und ausgeführt werden.

### 1. Einrichtung
1. Stelle sicher, dass die **Lesezeichenleiste** deines Browsers sichtbar ist (Tastenkombination: `Strg + Umschalt + B` bzw. `Cmd + Umschalt + B` auf dem Mac).
2. Erstelle ein beliebiges neues Lesezeichen in deiner Lesezeichenleiste (z. B. per Rechtsklick auf die Leiste -> *Seite hinzufügen* oder über den Lesezeichen-Manager).
3. Ändere den **Namen** des Lesezeichens in etwas Eindeutiges (z. B. `🔍 AdInspector`).
4. Kopiere den gesamten JavaScript-Codeblock aus dem Feld [unten](#bookmarklet-code) (wichtig: er muss mit `javascript:` beginnen).
5. Füge den kopierten Code in das Feld **URL** (oder *Adresse*) des Lesezeichens ein und speichere es ab.

### 2. Durchführung der Analyse (Benutzung)
1. Navigiere im Browser auf eine öffentliche Webseite, die du überprüfen möchtest.
2. Klicke in deiner Lesezeichenleiste auf das Lesezeichen `🔍 AdInspector`.
3. **Wichtig - Automatisiertes Scrolling:** Das Tool wartet kurz und übernimmt dann autonom die Steuerung. Es scrollt die Seite in Intervallen schrittweise nach unten. Dieser Vorgang ist zwingend erforderlich, um *Lazy-Loading-Slots* (Werbebanner, die erst beim Herabscrollen geladen werden) verlässlich zu aktivieren.
4. Bitte warte, bis das Skript das Seitenende erreicht hat und automatisch wieder an die Spitzenposition zurückspringt.
5. Nach Abschluss der Analyse blenden sich die **farbigen Rahmen (Overlays)** über den Werbeannern sowie die **verschiebbare Sidebar** am rechten Bildschirmrand ein. Über die Sidebar kannst du alle technischen Details einsehen und den strukturierten Datensatz über den Button `📥 Export JSON` für die KI-Pipeline herunterladen.
